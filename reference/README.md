# reference/ — read-only

Original **Typst** source of the theme this repo ports to Beamer. Kept here so
Claude Code can consult it offline. **Do not compile or edit these files** — they
are Typst, not LaTeX, and serve only as the visual/behavioural source of truth.

- `clean-lib.typ` — the `touying-quarto-clean` v0.2.0 theme implementation
  (colours, list markers, section/title-slide layout, fonts). The key file.
- `template-full.qmd` — the full Quarto demo exercising every feature.

Upstream: https://github.com/kazuyanagimoto/quarto-clean-typst
